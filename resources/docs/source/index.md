`GET api/users`

#### Query Parameters

Parameter | Status | Description
--------- | ------- | ------- | -----------
    `fields[users]=id,email` |  optional  | 顯示哪些欄位
    `filter[&#039;id&#039;,` |  optional  | 'name', 'email']=abc 包含那些欄位自段

    `append=fullname` |  optional  | 附加欄位
    `include=oauthProviders` |  optional  | 關聯資料
